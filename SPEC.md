# API 套餐查询助手 - 规格文档

## 1. 项目概述

- **项目名称**: api_quota_helper
- **类型**: Flutter Android 应用
- **核心功能**: 通过配置用户名和 API Key，实时查询套餐额度、到期时间、剩余用量、下次刷新时间，并支持小米桌面小组件展示

## 2. 功能列表

### 2.1 主应用功能
- **API 配置**: 输入用户名和 API Key（支持多种服务商：OpenAI、Claude、Gemini 等）
- **数据查询**: 获取套餐信息、到期时间、额度、已用/剩余量
- **刷新控制**: 显示下次刷新时间、倒计时、进度条
- **历史记录**: 本地保存查询历史
- **多账户管理**: 支持添加多个 API 账户切换

### 2.2 桌面小组件
- **小号组件 (2x2)**: 显示当前套餐名称 + 剩余额度百分比
- **中号组件 (4x2)**: 显示套餐名 + 额度进度条 + 到期日期
- **大号组件 (4x4)**: 完整信息展示 + 下次刷新倒计时
- **多种样式**: 简约版、渐变色版、进度条版

### 2.3 数据展示
- 套餐名称
- 到期时间（倒计时）
- 总额度 / 已用额度 / 剩余额度
- 下次刷新时间
- 额度使用进度条（百分比环形/条形）

## 3. 技术方案

- **框架**: Flutter 3.x
- **状态管理**: Riverpod
- **本地存储**: SharedPreferences + SQLite
- **网络**: Dio
- **小组件**: home_widget (Flutter 插件)
- **UI**: Material Design 3

## 4. 打包配置

- 应用包名: `com.apiapp.quota_helper`
- 小米应用商店兼容
- 支持 Android 6.0+

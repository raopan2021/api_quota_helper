## 1. 执行摘要

### 1.1 核心结论

**关于用户问题"上半年低、下半年高"的规律：**

这个规律**并非普遍规律**，而是因为：
- **政府采购周期**：客车(尤其是公交车)采购通常在下半年集中，尤其是Q3-Q4
- **财政年度因素**：地方政府和国企多在下半年完成采购计划
- **宇通客车的季节性**：宇通客车确实存在一定的季节性，但并非绝对

**注意**：这种季节性波动并不构成可靠的交易策略，因为：
- 股价已提前反映预期
- 市场情绪、大盘走势等因素影响更大
- 行业竞争格局变化会影响这种规律

**关于"上半年低、下半年高"的解释：**

宇通客车的股价确实在一定程度上受到行业周期影响：

- **一季度(1-3月)**：通常是淡季，春节影响，订单较少
- **二季度(4-6月)**：逐步恢复，但仍是相对淡季
- **三季度(7-9月)**：迎来旺季，政府采购启动
- **四季度(10-12月)**：全年最旺季节，业绩集中释放

**但需要注意：**
- 这种规律**不是绝对的**
- 股价往往**提前反应**，到下半年时可能已经上涨
- 新能源政策、宏观经济等因素影响更大
- 近年行业竞争加剧，规律有所弱化

## 4. 客车行业季节性分析

### 4.1 行业特性

| 特性 | 说明 |
|------|------|
| **客户群体** | 政府采购(公交公司、旅游公司)、企业 |
| **采购周期** | 多在下半年尤其是Q3-Q4 |
| **结算周期** | 订单到确认收入有时滞 |
| **季节因素** | 春节前后为淡季 |

### 4.2 季节性数据参考

**宇通客车历史季度营收占比(参考)：**
- Q1：约18-20%
- Q2：约22-25%
- Q3：约25-28%
- Q4：约30-35%

**这确实表明下半年(特别是Q4)是收入确认的旺季。**

### 4.3 但投资不能简单套用

⚠️ **重要提醒：**

1. **预期已反映**：市场往往提前反映这种预期
2. **宏观因素更重要**：大盘走势、政策影响更大
3. **行业在变化**：新能源转型、竞争格局变化
4. **个案差异**：不是所有客车股都遵循此规律

- 你知道这个规律，别人也知道
- 股价往往提前1-2个季度启动
- 可能错过涨幅或买在高位
- 行业变化可能打破旧规律

5. **应收账款风险**：政府采购回款周期长
